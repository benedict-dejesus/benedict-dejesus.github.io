Lesson 3: My ISD Portfolio

Project Name: POKEFOLIO

POKEFOLIO is a tribute to one of the first video games I played. This version uses the game as a UI to showcase portfolio highlights, but such ideas could have instructional design applications too.

Project Name: Instructional Design Compass (IDC)

Instructional Design Compass (IDC) is envisioned to be the first step to effective instructional design processes. IDC uses Socratic method to probe deeper into the problem or situation and utilizes Bloom's Taxonomy and Adult Learning Theory to output a robust guide for the lessons.

Project Name: Titan Simulator (Titan-Sim)

Titan Simulator is a gamified simulation built to give basic training for beginner crane operators. This mini game is easily integratable to an e-learning module. WARNING: Titan Simulator is a desktop-only app.

Project Name: Cooking Papa

Cooking papa is a non-AI-powered, dynamic culinary mastery module. At present, it only features common recipes, but the branching tech, functions, logic, and gamification elements are the key products.
